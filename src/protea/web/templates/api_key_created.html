SSE Transport (Claude Desktop)

Add to your Claude Desktop MCP configuration:

{
  "mcpServers": {
    "protea": {
      "transport": {
        "type": "sse",
        "url": "http://localhost:8081/sse",
        "headers": {
          "Authorization": "Bearer {{ api_key.plaintext_key }}"
        }
      }
    }
  }
}

Stdio Transport

      }
    }
  }
}
